# dance-studio-website
## Цели проекта

**Основная цель** - создание современного веб-сайта как центрального цифрового пространства студии, объединяющего студентов, тренеров и администрацию.

## Аналитика данных

На основе анализа данных 100 студентов получены следующие данные:

### Распределение по возрастам
График "Распределение людей по возрастам" в analysis_users.py

**Выводы по возрастной структуре:**
- Основная аудитория (85%) - 14-24 лет
- Подростковые группы составляют стабильные 12%
- Детские группы (3%) показывают потенциал для роста
- Пик популярности приходится на возраст 16-20 лет

### Популярность направлений
График "Популярность направлений танцев" в analysis_users.py

**Выводы по популярности:**
- Hip-hop и Choreo - явные лидеры (42% человек)
- High Heels и Jazz funk сохраняют стабильную популярность
- Наблюдается равномерное распределение по современным стилям
- Классические направления уступают современным стилям

### Распределение по стилям
График "Соотношение стилей танцев" в analysis_users.py 

**Выводы по распределению стилей:**
- Четкое разделение по возрастным предпочтениям
- Каждое направление находит свою целевую аудиторию
- Отсутствие доминирования одного стиля - все сбалансировано

### Возраст по направлениям
График "Средний возраст людей по стилям танцев" в analysis_users.py 

**Выводы по возрастному распределению:**
- Baby (4-5 лет) - строгое соблюдение возрастных ограничений
- Teens (10-14 лет) - переходная возрастная группа
- High Heels и Girly Choreo - старшая аудитория (16+)
- Hip-hop и Choreo - широкий возрастной охват (14-24 года)
- Jazz funk - средний возрастной диапазон (12-18 лет)

### Финансовая аналитика
График "Выручка по направлениям" в analysis_price.py 

**Выводы по финансовой эффективности:**
- Направления стоимостью 3000-4000 руб генерируют 65% выручки
- High Heels и Choreo - наиболее доходные направления
- Детские группы обеспечивают стабильный доход
- Премиальные направления (4000 руб) имеют потенциал роста

### Оптимизация расписания
График "Расписание тренеров" в analysis_schedule.py 

**Выводы по расписанию:**
- Четкое разделение временных слотов между тренерами
- Пиковая нагрузка в вечерние часы (19:00-21:30)
- Вторник и четверг - наиболее загруженные дни

## Преимущества сайта

### Профессиональное представительство
Собственный сайт создает официальный образ и вызывает доверие у людей, которые будут им пользоваться. К тому же, намного удобнее узнать всю информацию на сайте, чем ходить по разным соц. сетям в поисках нужной информации.
Все актуальные данные собраны в одном месте:
- Расписание
- Цены
- Направления  
- Тренеры

### Автоматизация работы
Снижение нагрузки на администраторов:
- Онлайн-запись вместо звонков
- Автоматическое обновление расписания
- Ответы на вопросы через FAQ
